# RiskGuard


RiskGuard is a machine learning application that predicts loan approval risk using applicant financial information.

## Features

- Random Forest prediction model
- Applicant CSV uploads
- Approval probability estimates
- Data preprocessing pipeline
- Interactive dashboard
- Income distribution visualization
- Correlation heatmap
- Confusion matrix evaluation

## Tech Stack

- Python
- Pandas
- Scikit-learn
- Matplotlib
- CustomTkinter
- Joblib

## Model Performance

- Accuracy: 89.1%
- Precision: 93.2%
- Recall: 59.6%
- F1 Score: 72.7%
